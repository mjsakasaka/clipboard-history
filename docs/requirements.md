# 开发需求文档

## 项目概述

**Clipboard History** — Windows 历史粘贴板管理软件。自动记录用户复制的内容（文字和图片），提供浏览、搜索、再次粘贴的功能。

## 目标用户

- 经常复制粘贴的办公人员
- 需要回溯历史复制内容的开发者
- 任何有剪贴板管理需求的 Windows 用户

---

## 功能需求

### F1 - 自动记录剪贴板

| 属性 | 描述 |
|------|------|
| 优先级 | P0（核心） |
| 描述 | 软件在后台运行期间，自动监测并记录剪贴板中的所有新内容 |
| 支持类型 | 纯文本、HTML 文本、图片（PNG/JPG/BMP/GIF） |
| 去重 | 连续复制相同内容不重复记录 |

**验收标准：**
- 复制一段文字后，软件能记录该文字
- 复制一张图片后，软件能记录该图片
- 连续复制相同内容，只保留一条记录

---

### F2 - 历史浏览

| 属性 | 描述 |
|------|------|
| 优先级 | P0（核心） |
| 描述 | 用户打开主窗口后，以卡片形式浏览所有剪贴板历史 |

**卡片展示规则：**
- 置顶的卡片永远排在最前面
- 同组内按时间降序排列（最新在前）
- 每张卡片显示：内容预览、时间戳、操作按钮

**文字卡片预览：**
- 单行最多显示 80 个字符，超出截断加 "…"
- 多行文本显示前 3 行

**图片卡片预览：**
- 显示缩略图，保持宽高比
- 缩略图统一高度 80px

**验收标准：**
- 打开主窗口，能看到所有历史记录
- 置顶卡片排在前面
- 图片显示缩略图

---

### F3 - 再次粘贴

| 属性 | 描述 |
|------|------|
| 优先级 | P0（核心） |
| 描述 | 点击卡片，将对应内容重新写入系统剪贴板 |
| 反馈 | 粘贴成功后显示"已复制"提示，窗口不关闭 |

**验收标准：**
- 点击文字卡片后，在其他应用中 Ctrl+V 能粘贴出对应文字
- 点击图片卡片后，在其他应用中 Ctrl+V 能粘贴出对应图片

---

### F4 - 置顶

| 属性 | 描述 |
|------|------|
| 优先级 | P1（重要） |
| 描述 | 用户可以将任意卡片置顶，使其始终显示在列表最前面 |

**交互：**
- 点击卡片上的 📌 按钮切换置顶状态
- 已置顶的卡片显示不同的图标颜色（实心）

**验收标准：**
- 置顶后卡片移到最前面
- 取消置顶后卡片恢复正常排序位置
- 刷新列表后置顶状态保持不变

---

### F5 - 删除

| 属性 | 描述 |
|------|------|
| 优先级 | P1（重要） |
| 描述 | 用户可以删除不需要的历史记录 |

**交互：**
- 点击卡片上的删除按钮
- 删除前弹出确认提示
- 图片类型同时删除对应的图片文件

**验收标准：**
- 删除后卡片从列表消失
- 刷新后确认已从数据库移除

---

### F6 - 搜索

| 属性 | 描述 |
|------|------|
| 优先级 | P1（重要） |
| 描述 | 用户可以通过关键词搜索文字类型的历史记录 |
| 匹配方式 | 模糊匹配（SQL LIKE） |
| 触发方式 | 实时过滤（输入即搜） |

**验收标准：**
- 输入关键词后列表实时过滤
- 清空搜索框后恢复完整列表
- 搜索不区分大小写（可选）

---

### F7 - 存储策略

| 属性 | 描述 |
|------|------|
| 优先级 | P1（重要） |
| 描述 | 自动清理过期和超量记录 |

**两条规则同时生效：**
1. **时间过期**：超过设定天数的非置顶记录自动删除
2. **数量上限**：超过设定最大条数时，删除最旧的非置顶记录

**清理频率：** 每小时检查一次

**验收标准：**
- 设置 1 天过期后，1 天前的记录被自动删除
- 设置最大 100 条，第 101 条写入后最旧的被删除
- 置顶的记录永远不会被自动清理

---

### F8 - 设置面板

| 属性 | 描述 |
|------|------|
| 优先级 | P2（一般） |
| 描述 | 用户可以自定义软件行为 |

| 设置项 | 默认值 | 范围 |
|--------|--------|------|
| 保存天数 | 3 天 | 1 / 3 / 5 |
| 最大条数 | 500 | 100 ~ 2000 |
| 启用全局快捷键 | 是 | 是 / 否 |
| 快捷键组合 | Ctrl+Shift+V | 用户自定义 |
| 开机自启 | 是 | 是 / 否 |

**验收标准：**
- 修改设置后立即生效
- 重启软件后设置保持不变

---

### F9 - 系统托盘

| 属性 | 描述 |
|------|------|
| 优先级 | P1（重要） |
| 描述 | 软件最小化到系统托盘运行 |

**右键菜单：**
- 打开历史面板
- 设置
- 暂停监控 / 恢复监控
- 退出

**验收标准：**
- 程序运行后托盘出现图标
- 右键菜单所有选项可用
- 点击"打开历史面板"弹出主窗口
- 退出后托盘图标消失

---

### F10 - 全局快捷键（可选）

| 属性 | 描述 |
|------|------|
| 优先级 | P2（一般） |
| 描述 | 用户可通过快捷键快速打开历史面板 |
| 默认 | Ctrl+Shift+V |
| 可配置 | 用户可在设置中修改或禁用它 |

**验收标准：**
- 按下快捷键后弹出主窗口
- 在设置中修改快捷键后立即生效
- 禁用快捷键后按键无反应

---

## 非功能需求

| 维度 | 要求 |
|------|------|
| 性能 | 监控轮询间隔 500ms，UI 操作响应 < 200ms |
| 内存 | 空闲时内存占用 < 50MB |
| 数据安全 | 数据库文件存储在本地用户目录，不联网 |
| 兼容性 | Windows 10 / Windows 11 |
